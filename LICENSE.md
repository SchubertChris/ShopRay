**Letzte Aktualisierung:** 2026-05-14

## Gewährte Rechte

Mit dem Kauf einer ShopRay-Lizenz erhältst du das nicht-exklusive, nicht-übertragbare Recht:

- Den Quellcode für **ein (1) kommerzielles Projekt** zu verwenden und zu deployen
- Den Code an deine eigenen Anforderungen anzupassen
- Den Code auf deiner eigenen Infrastruktur zu betreiben

## Was nicht erlaubt ist

- Den Code oder Teile davon **weiterzuverkaufen**, zu verleihen oder zu verschenken
- Den Code als eigenes Template-Produkt **weiterzuvertreiben**
- Die Lizenz auf eine andere Person oder ein anderes Unternehmen **zu übertragen**
- Den Code für **mehrere Projekte** ohne zusätzliche Lizenz zu nutzen

## Eigenverantwortung

- Du bist verantwortlich für die **Rechtssicherheit** deines Shops (Impressum, Datenschutz, AGB)
- Du bist verantwortlich für die korrekte **Supabase, Stripe und SMTP-Konfiguration**
- Der Lizenzgeber übernimmt keine Haftung für Schäden, die durch die Nutzung entstehen
